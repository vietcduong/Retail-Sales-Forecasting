**Retail Sales Forecasting Using Machine Learning**
Overview

This project analyses historical sales data from 1,115 retail drug stores in Germany to forecast daily sales for a six-week horizon. The objective was to identify key sales drivers and develop predictive models to support data-driven retail decision making.

The analysis combines exploratory data analysis, clustering, and predictive modelling using Python.

Business Problem

Retailers rely on accurate demand forecasts to optimise:

• inventory
• staffing
• promotions
• store operations

This project aims to build forecasting models that capture seasonality, promotions, and store characteristics.
